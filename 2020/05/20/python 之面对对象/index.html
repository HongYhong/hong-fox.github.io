继承与多态

在面向对象程序设计中，一个新的类可以继承自旧的类，这个新的类拥有旧的类的全部功能，新类成为子类，旧类称为父类或者超类。

类的继承可以方法覆盖。

定义一个class，也就相当于定义了一个数据类型。

多态：一个信息发送给不同的对象而产生不同的执行效果。信息的发送即函数的调用。

这样做的优点是，调用无需关注调用函数的细节，当我们添加新类时我们只需要确保run()方法的正确。

“开闭”原则：对外开放添加新的子类。

对内修改封闭：不需要修改依赖该类的函数。

动态语言 vs 静态语言

像Java这样的静态语言，如果函数要求传入某种类型那我们只能传入该类型或该类型的子类。但是像python这样的动态语言只需要传入的类型拥有函数所要求的方法即可。“鸭子类型”。file-like object，真正的文件类型拥有read()方法，file-like object只需要也带有read()方法即可。
